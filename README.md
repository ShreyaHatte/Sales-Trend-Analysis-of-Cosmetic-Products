This project analyzes cosmetics sales data using Python and Machine Learning to identify trends, top-performing categories, and customer behavior.

Key features:

Data cleaning and preprocessing using Pandas
Sales trend visualization using Matplotlib & Seaborn
Correlation analysis to understand relationships between variables
Linear Regression for predicting total sales (R² Score evaluation)
Classification model (Decision Tree) to categorize products into High/Low sales
Performance evaluation using Accuracy Score and Confusion Matrix

The project demonstrates how data analysis can help businesses make better decisions based on sales patterns.
